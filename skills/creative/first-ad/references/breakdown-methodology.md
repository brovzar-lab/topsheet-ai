## 1. Reading the Script as a 1st AD

The 1st AD does NOT read the script for story. You read it for logistics. On every pass:

- **Pass 1:** Count scenes. Number them sequentially (scene 1, 2, 3…). If the writer hasn't numbered scenes, you do it.
- **Pass 2:** Flag every scene element that costs money or time: cast, extras, locations, props, SFX, VFX, stunts, animals, vehicles, special rigs.
- **Pass 3:** Identify storyline continuity requirements — when does actor X need to look injured? When does the car need to be wrecked? Continuity in a scene can lock your shooting order.
- **Pass 4:** Group scenes by location. This is your primary scheduling unit.

Never break down from memory. Always go back to the page.

## 2. Scene Element Tagging

For every scene, tag the following elements:

### Cast (Principal)
- List all speaking characters by role
- Identify if character requires physical transformation (hair change, makeup, prosthetics, weight vest, aging)
- Note if character appears injured, dead, or in continuity with adjacent scenes

### Background / Extras
| Scale | Category |
|-------|----------|
| 1–9 | Few extras (no BG AD needed) |
| 10–49 | Featured extras — add crowd choreography time |
| 50–99 | Large BG day — add 1 BG AD, 30 min setup |
| 100+ | Major BG day — multiple BG ADs, 45–60 min setup, holding area required |

### Special Elements (flag these immediately)
- **Stunts** — Stunt coordinator required; safety meeting; rehearsal time; camera coverage implications
- **VFX Plates** — On-set VFX supervisor; witness camera; reference balls; lens data collection
- **SFX / Practical Effects** — Fire, rain, smoke, pyrotechnics, breakaways — each requires setup time, safety perimeter, permits
- **Water Work** — Any scene with wet actors, rain, pools, ocean, rivers — multiply time estimate by 1.5–2x
- **Animals** — Wranglers, SAGARPA/APA/REAG approvals (territory-dependent), limited working hours
- **Child Actors** — Regulated working hours by territory (see territories.md); requires guardian on set; education hours may apply; welfare rep
- **Vehicles** — Working/non-working, hero vehicle, insert car, process trailer — each adds logistics overhead
- **Drones** — Regulatory approval required; dedicated drone team arrival before crew
- **Prosthetics / Special Makeup** — Extended makeup calls (2–6 hrs extra); schedule actor's day accordingly

## 3. Eighths Calculation

The industry standard unit for measuring script length is the **eighth of a page**.

- 1 full page = 8/8ths
- ½ page = 4/8ths
- ¼ page = 2/8ths
- A slug line alone = 1/8th

**Why it matters:** Scheduling is done in eighths. A "2-page scene" can be anywhere from 12/8ths to 20/8ths depending on action density. Always measure exactly.

**Practical guide:**
- Measure each scene from the slug line to the last line before the next slug line
- Count in increments of 1/8th (one inch of script ≈ 1/8th page on industry-standard Final Draft formatting)
- Total all scenes; the sum = your total page count

**Sanity check:** A feature screenplay is typically 90–120 pages = 720–960/8ths. A TV drama hour-long is 45–65 pages.

## 4. Complexity Scoring System

Use this scoring to estimate time per scene, not just page count. Two 1-page scenes can be wildly different in shoot time.

### Base Time Estimate (from page count)
| Pages/day target | Time per page (10h day) |
|-----------------|------------------------|
| 3 pages/day | ~3.3 hrs/page |
| 4 pages/day | ~2.5 hrs/page |
| 5 pages/day | ~2 hrs/page |
| 6 pages/day | ~1.7 hrs/page |

### Complexity Adjustments

**REDUCE pages/day estimate when scene contains:**
- Stunt sequence: −0.5 to −1.5 pages/day (add safety meeting, rehearsal, medical standby time)
- Water (any): −1 to −2 pages/day (wet actor turnaround, safety, drying time)
- SFX element: −0.5 to −1 pages/day (reset time between takes is 3–15 min for fire/pyro)
- Large extras (100+): −0.5 pages/day (crowd movement takes multiple takes)
- Complex camera rig (crane, Technocrane, underwater housing): −0.5 pages/day (setup time)
- Night exterior: −0.5 to −1 pages/day (lighting setup takes longer; crew fatigue after 2 AM)
- Multiple company moves on same day: −0.5 pages/move
- Child actors in scene: applies local regulation caps to shooting window (see territories.md)
- Animal in scene: −0.5 to −1 pages/day (animal unpredictability; trainer setup)

**INCREASE pages/day estimate when scene is:**
- Single interior location, all dialogue, small cast: can push to 6–8 pages/day
- Music video-style sequence (same setup repeated): can shoot out of sequence efficiently
- Table reads or limited movement scenes

## 5. Location & Set Grouping

**Primary scheduling rule:** Always group scenes by location first. Every unnecessary company move is money wasted and time lost.

### Location Grouping Process
1. List every unique location in the script
2. Assign all scenes to their location
3. Total the page count per location
4. Identify interior/exterior for each location (different lighting setups)
5. Sort locations by total page count (heaviest first, unless cast availability or continuity overrides)

### Set vs. Location
| Type | Notes |
|------|-------|
| Stage/Studio set | Maximum control; schedule when weather protection is priority |
| Practical interior | Subject to owner availability, sound issues; scout early |
| Practical exterior | Weather-dependent; always have interior cover set ready |
| Remote location | Add company move time + logistics overhead; assess proximity to base camp |

### Cover Set Rules
- Always designate 1–2 interior scenes as cover sets for weather-dependent exterior days
- Cover set must be prep-ready (art department done, lighting plan pre-positioned)
- Never leave a day without a cover option unless the exterior scene can move 1:1 to another day without breaking continuity

## 6. Cast Day Analysis

### DOOD (Day Out of Days) Chart

The DOOD maps every principal cast member across every shooting day:

| Symbol | Meaning |
|--------|---------|
| **W** | Work — shooting this day |
| **H** | Hold — contracted, on location, not shooting |
| **T** | Travel |
| **F** | Finish — last day on schedule |
| **SW** | Start and Work (first day) |
| **SWF** | Start, Work, and Finish (one day only) |

**Goal:** Convert holds to drops. Every hold day = salary paid for zero production value.

### Cast Availability Windows
- Identify any cast member with a hard out date (another project, delivery commitment, visa expiry)
- Flag cast with multiple productions — turnaround and rest reqs still apply
- Note cast requiring action preparation: stunt training, dialect coaching, physical transformation — these happen in prep, before shooting begins

### Child Actor Windows (General)
- Hours on set are regulated by territory (see territories.md)
- Always have welfare rep/guardian check-in before calling child to set
- Build call time to allow for school or tutoring hours if required

## 8. Common Breakdown Mistakes

1. **Counting slug lines without reading the scene** — A slug line for "INT. HOSPITAL — CONTINUOUS" that opens into a 3-page surgical sequence is not a simple interior.

2. **Missing coverage of brief scenes** — A 1/8th page scene with the lead actor at a specific location still requires full travel, setup, and coverage time. Don't assume short = fast.

3. **Ignoring script days** — Scenes written as DAY 3 of story time may require continuity elements (wounds, wardrobe, prop states) that lock scheduling order.

4. **Under-flagging extras** — "A busy café" reads like 5 extras. On screen it requires 40. Always cross-check location description against script intent.

5. **Forgetting turnaround when scheduling nights after days** — If you wrap Day 10 at 1 AM and call Day 11 at 7 AM, you've violated 12-hour turnaround for the entire crew.

6. **Treating all dialogue scenes as equal** — A 2-page two-hander in a kitchen is different from a 2-page courtroom scene with 6 principals, jury box BG, and judge reaction cutaways.

7. **Not flagging hidden VFX** — "The city skyline glitters behind them" = VFX extension. Productions regularly miss these until post.
